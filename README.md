# ♟️ ChessMate

**Master the game, one opening at a time.**

ChessMate is an interactive learning platform focused on helping users understand and master chess openings. Currently, it features the **London System** and **Caro-Kann Defense**, with clean UI and responsive design.

## 🚀 Stack
- React
- TailwindCSS
- React Router
- GitHub Pages or Vercel (Deployment)

## 🧠 Goal
Designed to be a GitHub portfolio project for job applications.

## 📦 Setup
```bash
git clone https://github.com/your-username/chessmate.git
cd chessmate
npm install
npm start
